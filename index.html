Taller sobre Grafos

1. ¿Qué es un grafo?

Un grafo es una estructura matemática que representa relaciones entre pares de objetos. Está compuesto por nodos (o vértices) y aristas (o bordes) que conectan estos nodos. Los grafos se utilizan para modelar relaciones en diversos contextos, como redes, mapas, o sistemas de comunicación.

2. ¿Para qué se utilizan?

Los grafos se utilizan para:

  • Modelar redes (sociales, de computadoras, de transporte).
  • Resolver problemas de optimización (como rutas más cortas).
  • Representar relaciones jerárquicas o dependencias (árboles genealógicos, organigramas).
  • Analizar estructuras complejas en ciencias de datos, inteligencia artificial, y bioinformática
  • Diseñar algoritmos para búsqueda, planificación, y recomendación.

3. Partes y tipos de grafos

Partes de un grafo:

  • Nodos (vértices): Representan entidades (ejemplo: ciudades, personas).
  • Aristas (bordes): Representan relaciones o conexiones entre nodos (ejemplo: carreteras, amistades).
  • Peso (opcional): Valor asociado a una arista (ejemplo: distancia, costo).
  • Dirección (opcional): Indica si la relación es unidireccional o bidireccional.

Tipos de grafos:

  • Dirigido (digrafo): Las aristas tienen dirección (A → B no implica B → A).
  • No dirigido: Las aristas son bidireccionales (A — B implica B — A).
  • Ponderado: Las aristas tienen pesos (ejemplo: distancias).
  • No ponderado: Las aristas no tienen pesos.
  • Cíclico: Contiene al menos un ciclo (camino que regresa al mismo nodo).
  • Acíclico: No contiene ciclos.
  • Conexo: Existe un camino entre cualquier par de nodos.
  • No conexo: Algunos nodos no están conectados.

4. Campo de aplicación

Los grafos se aplican en problemas reales como:

  • Redes de transporte (rutas óptimas)
  • Redes sociales (conexiones entre personas)
  • Redes de computadoras
  • Biología (análisis de genes y proteínas)
  • Algoritmos de búsqueda y recomendación

5. Ejemplos en Python

· BFS (Breadth-First Search) – Búsqueda por amplitud


from collections import deque

def bfs(grafo, inicio):
    visitados = set()
    cola = deque([inicio])

    while cola:
        nodo = cola.popleft()
        if nodo not in visitados:
            print(nodo)
            visitados.add(nodo)
            cola.extend([v for v in grafo[nodo] if v not in visitados])

grafo = {
    'A': ['B', 'C'],
    'B': ['D', 'E'],
    'C': ['F'],
    'D': [],
    'E': ['F'],
    'F': []
}

bfs('A', grafo)

· DFS (Depth-First Search) – Búsqueda por profundidad


def dfs(grafo, nodo, visitados=None):
    if visitados is None:
        visitados = set()

    if nodo not in visitados:
        print(nodo)
        visitados.add(nodo)
        for vecino in grafo[nodo]:
            dfs(grafo, vecino, visitados)

grafo = {
    'A': ['B', 'C'],
    'B': ['D', 'E'],
    'C': ['F'],
    'D': [],
    'E': ['F'],
    'F': []
}

dfs(grafo, 'A')

Estos algoritmos permiten recorrer grafos de diferentes maneras y se utilizan para resolver problemas como encontrar caminos, detectar ciclos, entre otros.
